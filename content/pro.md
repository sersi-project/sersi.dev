## Get Access

1. Purchase a license at [sersi.dev/pro](https://sersi.dev/pro)
2. Receive an API token and Pro CLI access
3. Authenticate via:

```bash
sersi-pro init --token=YOUR_API_TOKEN
```

## Usage

```bash

sersi-pro build -f path/to/config.yaml
sersi-pro create <flags>
```
